# Inshar Ul Haq & Company — Website

A responsive real estate showcase website for **Inshar Ul Haq & Company**, a family-run luxury home builder with 40+ years of experience in Islamabad, Pakistan.

## Overview

The site presents the company's current property listing with a photo gallery, video walkthroughs, full construction specifications, and an enquiry form. It's designed to give buyers a complete feel for each home and make it easy to get in touch.

## Features

- Rotating hero section with cinematic imagery
- Two-page structure: homepage + dedicated property detail page
- Photo gallery with click-to-enlarge lightbox (29 images)
- Embedded video clips of the property
- Enquiry form connected via Formspree
- Fully responsive with a mobile navigation menu

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- [Formspree](https://formspree.io/) for form handling
- Deployed on [Netlify](https://www.netlify.com/)
